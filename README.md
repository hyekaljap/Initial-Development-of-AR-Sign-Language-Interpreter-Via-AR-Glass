# AR Sign Language Interpreter via AR Glasses
### PSM 1 — Initial Development

## Project Overview
A real-time sign language interpretation system using Augmented Reality (AR) glasses to assist communication for the deaf and hard-of-hearing community.

## Objectives
- Detect and classify sign language gestures using computer vision
- Display real-time text/translation overlay via AR glasses
- Achieve low-latency inference suitable for live interaction
